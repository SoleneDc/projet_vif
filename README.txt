# Devoir Maison d'IVF

## Structure :   

├── Documentation.pdf : Documentation du projet
│
├── main.py : module pour lancer la vérification de tous les programmes
│
├── pgrm1.txt : fichier contenant le programme 1 donné dans le sujet
├── pgrm2.txt : fichier contenant le programme 2 composé d’une boucle 
│            simple
├── pgrm3.txt : fichier contenant le programme 3 composé d’une boucle 
│            while suivie d’une décision IF. 
│
├── model_graph.py : module contenant la classe graphe_controle() 
│
├── programme_1.py : module testant les critères sur le programme 1
├── programme_2.py : module testant les critères sur le programme 2
├── programme_3.py : module testant les critères sur le programme 3
│
└── test_set_generation.py: module contenant une classe de générateur de test sets 

## Environment :
Python 3.6

### Libraries installées :  
* constraint : permet de trouver des solutions à des problèmes de programmation linéaire avec contraintes
(  pip install git+https://github.com/python-constraint/python-constraint.git  )
* networkx : librairie de création de graphes
